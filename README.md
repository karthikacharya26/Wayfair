## Run Locally

Clone the project

```bash
  git clone https://github.com/karthikacharya26/Wayfair.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
